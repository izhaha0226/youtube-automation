# 자막 번역 프롬프트

## 역할
한국어 나레이션을 {{target_lang_name}}로 자연스럽게 현지화 번역하는 자막 전문가.

## 입력
- 문장 목록(KO): {{sentences_json}}

## 규칙
- 직역보다 현지 시청자가 바로 이해하는 현지화
- 문장 길이는 원문 문장 단위 유지 (타이밍 매칭 위해)
- 경제/부동산 용어는 현지 표준 용어로 통일
- 숫자·고유명사는 현지 관례대로

## 출력 (JSON만, 배열 길이는 입력과 동일)
```json
{
  "lang": "{{lang_code}}",
  "sentences": ["번역1", "번역2", "..."]
}
```

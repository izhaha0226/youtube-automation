# 나레이션 변환 프롬프트

## 역할
시나리오를 성우가 읽을 자연스러운 한국어 나레이션으로 변환.

## 입력
- 시나리오: {{scenario_json}}
- 예상 길이(초): {{expected_length_sec}}
- 톤: {{tone}}

## 규칙
- 말하는 문장, 호흡 자연화
- 한 문장 20~45자 내외. 너무 짧게 쪼개서 전체 분량을 줄이지 마라.
- 쉬운 단어
- 강조어는 *굵게* 표시
- 리치고 톤
- 예상 길이가 600초 이상이면 전체 text_ko는 공백 포함 최소 4,500자 이상이어야 한다.
- 뉴스 근거가 있는 시나리오는 단순 요약하지 말고, 각 근거마다 “뉴스 내용 → 시장 해석 → 실수요자 예시 → 판단 기준”으로 풀어서 말한다.
- scenario의 body_sections.narration을 삭제하거나 축약하지 말고, 더 자연스럽게 이어 붙이고 필요하면 예시를 보강한다.

## 출력 (JSON만)
```json
{
  "text_ko": "전체 나레이션",
  "sentences": ["문장1", "문장2", "..."],
  "emphasis": ["강조어1", "..."]
}
```

# 나레이션 변환 프롬프트

## 역할
시나리오를 성우가 읽을 자연스러운 한국어 나레이션으로 변환.

## 입력
- 시나리오: {{scenario_json}}
- 예상 길이(초): {{expected_length_sec}}
- 톤: {{tone}}

## 규칙
- 말하는 문장, 호흡 자연화
- 한 문장 20자 내외
- 쉬운 단어
- 강조어는 *굵게* 표시
- 리치고 톤

## 출력 (JSON만)
```json
{
  "text_ko": "전체 나레이션",
  "sentences": ["문장1", "문장2", "..."],
  "emphasis": ["강조어1", "..."]
}
```

# 업로드 메타 생성 프롬프트

## 역할
유튜브 업로드 메타(제목/설명/태그/해시태그/고정댓글)를 작성.

## 입력
- 주제: {{topic}}
- 시나리오: {{scenario_json}}
- 채널: 리치고
- 김기원 대표 철학: {{kim_kiwon_philosophy}}
- 리치고TV 편집 원칙: {{editorial_rules}}

## 규칙
- 제목: 60자 이내, 검색성 + 클릭성
- 설명: 3단락 구성 (요약 / 본문 포인트 / CTA)
- 태그: 15개 이내, 관련도 높은 순
- 해시태그: 3~5개
- 고정댓글: 댓글 유도 문구
- 메타 전체가 단순 낚시보다 판단형, 실수요형, 리치고다운 해석 톤을 유지해야 한다.

## 출력 (JSON만)
```json
{
  "title": "...",
  "description": "...",
  "tags": ["...", "..."],
  "hashtags": ["#...", "#..."],
  "pinned_comment": "..."
}
```
